# Bank

## Requirements

- interact with the your code via a REPL like IRB or the JavaScript console.
- allows deposits and withdrawals transactions.
- Account statement (date, amount, balance) printing.
- Data can be kept in memory.

## Getting started

This program uses Ruby. Install ruby before starting.

$ gem install bundler

$ bundle install

$ git clone https://github.com/pedrocastanheira77/bank.git

## Usage

Use ruby REPL irb to interact.

$ irb

Don't forget to require './lib/account'.

## Running tests

$ rspec
